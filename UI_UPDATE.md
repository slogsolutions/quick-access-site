## Installation

### 1. Install Tailwind CSS Dependencies

```bash
cd Frontend
npm install tailwindcss postcss autoprefixer
```

### 2. Start Development Servers

**Backend:**
```bash
cd Backend
npm run dev
```

**Frontend:**
```bash
cd Frontend
npm run dev
```

## New Admin Features

### Register Users
1. Login as admin
2. Click "Register User" button in header
3. Fill in user details (username, email, password, role)
4. User is created and can immediately login

### User Roles
- **Admin**: Full access, can register users, see all categories
- **Employee**: Employee + Other categories
- **Marketer**: Marketing + Other categories
- **HR**: HR + Other categories
